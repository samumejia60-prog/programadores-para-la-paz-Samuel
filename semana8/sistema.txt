Comando 'uname -a': Muestra la información completa del sistema operativo y del núcleo (kernel) de Linux, incluyendo el nombre del host, la versión del kernel, la arquitectura del procesador (por ejemplo, x86_64 o aarch64) y la fecha de compilación.

Comando 'whoami': Muestra de forma directa el nombre de usuario de la sesión actual que está ejecutando los comandos en la terminal.

Importancia en la administración de servidores: Conocer esta información es fundamental para un administrador, ya que permite verificar si el sistema cuenta con los parches de seguridad adecuados para su versión específica de kernel, identificar la arquitectura antes de instalar software o paquetes compatibles, y auditar los privilegios del usuario actual para saber si se cuenta con permisos de superusuario (root) para realizar tareas administrativas críticas.
